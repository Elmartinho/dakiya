    Dakiya

    Postman Collections Convertor


    • Features
    • ✔ Secure: All your collections are not uploaded to server.
    • Dakiya runs in your browser.

    • ✔ Add postman environment collection and dakiya replaces the variables in the script.

    • ✔ Convert to artillery scripts.

    • ✔ Convert to gatling scripts. (coming soon)

    • ✔ Convert to jmeter scripts. (coming soon)

    • Getting Started
    • • Upload Postman Environment collection.(optional)
    • • Upload Postman script collection.(mandatory)
    • • Select the tool to convert.
    • • Download the converted script.


    Note: Dakiya only supports PostMan collections v2 and upwards.
